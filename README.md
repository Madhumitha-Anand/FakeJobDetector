# Fake Job Posting Detector

A Machine Learning web app that detects fraudulent job postings using NLP.

## Features
- ML-based fraud detection
- Interactive UI using Streamlit
- Displays model accuracy and prediction confidence

## Tech Stack
- Python
- Scikit-learn
- TF-IDF
- Logistic Regression
- Streamlit
